# LookTwice | Personal Landing Page

A streamlined, high-impact personal landing page for **Johnathan**. This version of the site utilizes a bold, structured grid system to balance heavy graphic elements with clear personal philosophy.

## 🧱 The Architecture
The site is built on a "Thesis-to-Manifesto" arc:
1.  **The Hero (Split-Grid)**: A 50/50 balance between the core identity statement and the red engraving graphic.
2.  **The Anchor (Manifesto)**: A bold "On Intention" section that acts as a narrative bridge.
3.  **Identity & Credo**: A staggered layout featuring a micro-bio followed by a grid of personal principles.
4.  **The Shadow Effect**: Every credo statement is paired with a delayed "Shadow Thought" that fades in as the user scrolls.

## 🎨 Design Principles
- **Structured Symmetry**: Uses a balanced split-grid to give the text and the engraving equal weight.
- **Atmospheric Depth**: A massive, fixed "Ghost Text" background at 2.5% opacity that creates a parallax effect during scrolling.
- **The "Second Look" Interaction**: Content is revealed using an `IntersectionObserver`. Black text appears first, followed by the red `#8C2B2B` shadow thought a half-second later.
- **Tactile Minimalism**: Inspired by premium print media, utilizing off-white "paper" backgrounds and high-contrast typography.

## 🚀 Quick Start
1.  **File Management**:
    Ensure `index.html`, `styles.css`, and your image file (`Adobe Express - file.jpg`) are all in the same folder.
2.  **Linking**:
    The code is pre-configured to link to Fontshare for the *Bespoke Serif* and *Supreme* font families.
3.  **Launch**:
    Open `index.html` in any modern web browser.

## 🛠️ Technical Stack
- **Typography**: Bespoke Serif (Headers) & Supreme (Body/Sans).
- **Styling**: Vanilla CSS3 with `mix-blend-mode: multiply` for seamless image integration.
- **Animation**: CSS Keyframes for the hero line and JavaScript for scroll-triggered entry.
